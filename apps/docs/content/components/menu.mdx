The `Menu` component provides a flexible and accessible way to create menus. It allows users to interact with various options and actions, presenting them in a hierarchical structure.

#### Primitives
- `Button`: Used to trigger the opening of the menu.
- `Menu.Content`: Wraps the content of the menu, including sections and items.
- `Menu.Section`: Represents a section within the menu, grouping related items.
- `Menu.Label`: Provides a label for a menu section.
- `Menu.Item`: Represents an actionable item within the menu.
- `Menu.Shortcut`: Displays shortcut keys for menu items.
- `Menu.Separator`: Separates different sections within the menu.
