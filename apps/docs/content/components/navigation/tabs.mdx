The `Tabs` component enables users to navigate between different sections of content within the same space.

#### Primitives
- `Tabs`: Represents the tabs container component.
- `Tabs.List`: Container for the tab list.
- `Tabs.Tab`: Represents an individual tab within the tab list.
- `Tabs.Panel`: Represents the content panel associated with a tab.
