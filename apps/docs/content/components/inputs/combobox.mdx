### Custom Value

The default `ComboBox` component does not allow custom values. You can enable this by setting the `allowsCustomValue` prop to `true`.
This will then stop the `ComboBox` of resetting the value on `blur`, to the current selected value.

### Loading Async Data

A combobox can be shown as loading by setting the `isLoading` prop to `true`.
This can be combined with the useAsyncList hook from react-stately, to make async data loading super simple.
