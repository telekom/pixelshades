The `Dialog` component provides a versatile way to create modal dialogs or pop-up windows. It allows for interactive content such as forms, messages, or any other type of user input. 

#### Primitives
- `Button`: Used to trigger the opening of the dialog.
- `Dialog.Trigger`: Wraps the button that triggers the dialog.
- `Dialog`: The main dialog container.
- `Dialog.Content`: Content area within the dialog, typically containing header, body, and footer sections.
- `Dialog.Header`: Container for the header section of the dialog.
- `Dialog.Title`: Title of the dialog.
- `Dialog.Footer`: Container for the footer section of the dialog.
- `Input`: Input fields for user input.
- `Label`: Labels for input fields.
