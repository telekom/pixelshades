## Examples

### Format Options

You can change the format of the number field by passing the `formatOptions` prop.
It supports the format options provided by the browser native [Intl.NumberFormat](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Intl/NumberFormat/NumberFormat).
