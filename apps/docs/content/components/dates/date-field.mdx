## Options 

### Before and After

You can add a before or after slot to a DateField. This can be used to add icons or other elements to the DateField.
It has a predefined size of `size-4`, that can be overwritten.

<ComponentPreview name="date-field/before-after" />

### Loading 

You can use the `isLoading` prop to show a loading spinner.

<ComponentPreview name="date-field/loading" />

### Granularity

You can change the granularity of what the DateField can display as seen here

<ComponentPreview name="date-field/granularity" />


### Hour Cycle

You can change the hour cycle of the DateField by passing the `hourCycle` prop.

<ComponentPreview name="date-field/hour-cycle" />

### Error Message

You can display an error message by passing the `errorMessage` prop. This message will show when invalid is set to `true`.

<ComponentPreview name="date-field/error" />

### With Label

You can add a label to the DateField by passing the `label` prop.

<ComponentPreview name="date-field/with-label" />
