## Options

### Variants

You can use the `variant` prop to change the style of the button.

It default to the `default` variant.

### Before and After

You can add a before or after slot to a button. This can be used to add icons or other elements to the button.
It has a predefined size of `size-4`, that can be overwritten.

<ComponentPreview name="button/before-after" />


### Links

You can use the `href` prop to create a link button.

<ComponentPreview name="button/link" />


### Loading 

You can use the `isLoading` prop to show a loading spinner.
