### Placement

The Drawer can appear from any edge of the screen. Pass the `placement` property and set it to `top`, `right`, `bottom`, or `left`.

<ComponentPreview name="drawer/directions" />


### Hide Close Button

The Drawer can be shown without the close button by setting the property `hideCloseButton` to `true`. Default is `false`.
