The `Table` component is used to display tabular data in a structured format. It provides a convenient way to organize and present data in rows and columns.

#### Primitives
- `Table`: Represents the table component.
- `Table.Header`: Container for table header content.
- `Table.Column`: Represents a column in the table header.
- `Table.Body`: Container for table body content.
- `Table.Row`: Represents a row in the table.
- `Table.Cell`: Represents a cell within a table row.
