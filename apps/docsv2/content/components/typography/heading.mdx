You can nest headings with the `HeadingLevel` component to create a hierarchy of headings.
Each heading level will decrease the level of the heading by one.

<ComponentPreview name="heading/preview" />

## Usage

### Flat heading levels

Alternatively, you can use the `Heading` component directly to create a heading with a specific level by passing the `level` prop.

The `Heading` defaults to a level of `1` if no `level` prop is passed.

<ComponentPreview name="heading/flat-heading-levels" />

### Nested heading levels

You can use the `level` prop on the `HeadingLevel` component to create a nested heading level aswell.
